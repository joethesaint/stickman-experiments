
Welcome to the stickman cinematic universe. Here you'll find games, 2D animations, and 3D implementations of our humble stick hero.

### 🚀 What's Inside?
*   `stickman_2d/`: A 2D web animation sketch.
*   `stickman_3js/`: A 3D iteration built with Three.js.
*   `stickman_game/`: A fully playable web game.

### 🛠️ How to Spin Up
Since all of these are web-based, the instructions are identical! Just pick a folder, fire up a server, and play:
```bash
cd stickman_game  # or stickman_2d or stickman_3js
python3 -m http.server 8000
```
Open `http://localhost:8000` to interact!
